## 📖 各学年课程概览

### 🟢 大一（基础夯实阶段）
| 课程名称 | 内容方向 |
|---------|---------|
| 电磁学   | 静电场、磁场、电磁感应、麦克斯韦方程组基础 |
| 高等数学 | 微积分、级数、微分方程等数学工具 |
| 热学     | 热力学定律、气体动理论、熵与热机 |
| 数物实验 | 基础物理实验方法与数据分析 |
| 线性代数 | 矩阵运算、特征值、向量空间及其物理应用 |

### 🔵 大二（专业核心过渡阶段）
| 课程名称 | 内容方向 |
|---------|---------|
| 电力力学 | 电场与力学结合，电荷运动与力场分析 |
| 电工技术 | 电路分析、交流/直流电路、变压器与电机基础 |
| 概率论与数理统计 | 随机变量、分布、统计推断与误差分析 |
| 光学     | 几何光学、波动光学、干涉衍射与偏振 |
| 经典力学 | 牛顿力学、拉格朗日与哈密顿体系 |
| 数物方法 | 数学物理方程、特殊函数、积分变换 |
| 原子物理 | 原子结构、光谱、量子初步与能级跃迁 |
